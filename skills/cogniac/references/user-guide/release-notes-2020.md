# Release Notes — 2020

## December, 2020

New feature

- Added Current IP field for GigE camera ![](../../../assets/media/images/image-1700612073986.png)

Enhancements

- Links for media id, filename, and domain unit from feedback media info, fullscreen view, and search view will open in a new tab.
- Points in the point detection app and point count app can be dragged to reposition.
- Changed area detection v2 delete key to "c".
- Application type configuration for Area Detection v2.
- Max auto-zoom for focus mode can be set using hotkey ‘t’ inbox detection, point detection, and point count feedback.

## November, 2020

New feature

- Added asset tags, managers, and last boot time to EdgeFlow

![](../../../assets/media/images/image-1700612388359.png) ![](../../../assets/media/images/image-1700612451682.png)

- Added GigE camera features to the network camera ![](../../../assets/media/images/image-1700612230098.png)
- Users can filter out detections below the detection threshold in the feedback view.

  ![](../../../assets/media/images/image-1700613133835.png)

Enhancements

- Access controls for EdgeFlow configuration changes.
- App copy will be limited to trainable apps.
- Editing output subjects will be disabled for network cameras, camera capture, and trainable apps.
- Upgrades to integration app code editor
- Migrate custom config to application config for all application types
- Support multiple DNS in EdgeFlow network configuration

## October, 2020

New feature

- The login interface changed to support third-party authentication for users with this requirement.
- We introduced a new application type, camera capture v2, which is supported by EdgeFlow EF-20-10-07b and above. ![](../../../assets/media/images/image-1700612643188.png)

New features
