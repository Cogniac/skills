# Invitation to Tenant

## Overview

To add a new user to a tenant, the tenant administrator must send an invitation to the user, whether they already exist.

Click on “Add to Tenant”.

![](../../../assets/media/images/image%2842%29.png)

Enter the email and the role of the user. Click on “Invite”.

![](../../../assets/media/images/image%2844%29.png)

Users can then accept or decline invites. Once an invite is accepted, the user is automatically added to the tenant.
