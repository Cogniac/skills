# I keep getting errors

****I keep getting errors - what am I doing wrong?****

Please analyze the HTTP response closely - the HTTP error code, the message, and the unique Cogniac error code at the end of the message string. You can find detailed error messages if you can match the unique error code to the Error Code section of our API Documentation (/docs/api-errors).

Also, check the API status page <https://cogniac.statuspage.io/> and @CogniacHQ to confirm that the API is online and functioning properly.

If you still have issues, please contact support@cogniac.ai with a detailed summary, and we'll quickly follow up!
