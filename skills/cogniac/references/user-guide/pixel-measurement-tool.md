# Pixel Measurement tool

### ****Overview:****

This feature was created to enhance the application by introducing pixel measurement capabilities within the feedback queue. The goal was to enable users to perform basic pixel-based measurements—such as calculating line lengths directly on images without requiring external tools.

![](../../../assets/media/images/image%28247%29.png)

### ****Feature Description:****

Users can select two or more points on an image to measure distances (in pixels). This function assists users and feedback teams (annotators) in verifying spatial accuracy within annotated images.
