## April 18, 2023

- New feature: Area focus for classification apps. We have added support for area focus in our classification apps. Using an area detection app as an upstream for a classification app, the output of area mask detection will be converted into area focus, providing more detailed prior information to the classification app. This helps to improve detection accuracy and overall performance of the classification app. ![](../../../assets/media/images/image-1700517247147.png) ![](../../../assets/media/images/image-1700517429756.png)
