# Data Retention

****Data Retention****

****How long does Cogniac retain customer data****

Cogniac gives customer complete control of how long their data is retained.

Any customer uploaded data ("media" in the Cogniac system) which is not used for continuous improvement of the models is subject to expiration settings which can be configured by the customer.

Each subject can be configured with a "Media Expiration Time" in the subject settings as seen here:

![A screenshot of a computer  Description automatically generated](../../../assets/media/images/data-retention-image-qmh9e3jo.png)

Setting the expiration time removes non-consensus media from the subject after the set length of time.

When non-consensus media has expired from all subjects, it will be automatically deleted from the system.
