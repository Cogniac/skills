# Release Notes — 2022

## December 21, 2022

- The ability to download media with overlayed detections was added. ![](../../../assets/media/images/image-1700519143545.png)

## November 22, 2022

- Added support for Meraki Gen3 (MV63, MV93 generations)
- You can also view the [Meraki AI app creation flow](/docs/meraki-ai-quick-start-guide)

## November 14, 2022

- A user-configuration option to automatically create new workflows whenever CloudCore apps change ![](../../../assets/media/images/image-1700521786552.png)

## September, 2022

- We have added a new template wizard for creating a Meraki Camera box detection app.
- You can find the full document [here](/docs/meraki-api-advanced-setup-guide)

## August, 2022

Enhancements

- The new navigation bar groups the various items/links under a singular drop-down for a more intuitive navigation experience. ![](../../../assets/media/images/image-1700522028232.png)
- The Application Builder is introduced on the left-hand side of the home screen. Use this to upload media, create subjects, and create applications. Hovering over an item in the builder and its info icon shows a visual representation and/ or description. Clicking on the blue Application Builder tab contracts/ expands the builder.

- Moved the list view of applications and subjects to appear when clicking in the search box. ![](../../../assets/media/images/image-1700526978000.png)
- Moved the Release Notes under Usage -> Release Notes ![](../../../assets/media/images/image-1700522260640.png)

## April, 2022

Enhancements

- Support versioned workflows to lay the groundwork for automatically creating "latest" style workflow versions (e.g., on model release) and simplify workflow management. ![](../../../assets/media/images/image-1700527020613.png)

## March, 2022

Enhancements

- Added support for multiple versions of Python in the Integration app: 2.7, 3.6, and 3.10
- The training credit policy for model-based applications is modified to award 1 credit for each new media (or media-focused) labeled to correct the inference result on the media. ![](../../../assets/media/images/image-1700527115241.png)

New feature

- Added a new application event-type, training credit, to report the number of training credits rewarded
