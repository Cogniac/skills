# Tenant Expiration

****Tenant Expiration notice:****
Tenant users can now check whether their contract will expire soon and renew it to continue using the tenant as usual. A message informs the tenant Users that the contract is nearing its expiration date. Once the message appears, someone from our Sales representatives will contact the Tenant Admin for further discussion.![](../../../assets/media/images/tenant-expiry-image-gzau6nt6.png)

Users will be able to check when their contract expires by selecting the tenant Settings page. They need to hover over the Username and select Settings from the menu.
![](../../../assets/media/images/image%28183%29.png)
After that, they would need to scroll down, and they will be able to see the details displayed, such as the Contract End Date and some of the tenant limitations (if there are any limitations).![](../../../assets/media/images/image%28182%29.png)

****What has changed:****

When a tenant has expired, users will no longer have access to it. Attempts to visit the expired tenant will be blocked, ensuring it cannot be interacted with. The tenant icon will also appear grayed out in the interface, visually indicating its expired status.

![](../../../assets/media/images/tenant-expiry-image-n5osudfu.jpg)

> NOTE:
>
> Once a tenant subscription has expired, users will have only read access to the tenant's content (Media, assertions to media, etc.) using the SDK.
