# Release Notes

## February 18, 2026

### Deployed Workflow Sync Visibility

### Improvement

Deployed workflows now display sync status and divergence details directly on the ****Workflows List**** page.

Deployed workflows are shown at the top of the page and include a sync indicator to show whether their apps match the CloudCore versions clearly.

---

### What’s Improved

- Clear visibility into which workflows are currently deployed.
- Visual indicator showing whether apps are ****in sync**** or have ****diverged****.
- Ability to view a list of diverged apps directly from the Workflows page.
- Option to update and deploy the workflow when divergence is detected.
- When the button ****Update and Deploy Workflow**** is selected, the system confirms the changes, creates a new workflow version, and initiates deployment through the process.
- Updates the workflow status to ****“Deployed to: <Deployment\_Group\_Name>”**** after a successful Deployment Group update and deployment.
