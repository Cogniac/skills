# Tenant Home

## Tenant Homepage

By default, when users access a Tenant they will see the pipeline view. If the tenant has applications, this view will display a graph showing the application and subject nodes. Users can navigate to an application or subject by clicking on the corresponding node.

![](../../../assets/media/images/image%2857%29.png)

## Subjects view

When a subject node represents multiple subjects, clicking it will show a searchable list of subjects. Users can click on the required subject to take them to the subject view.

![](../../../assets/media/images/image%2858%29.png)
