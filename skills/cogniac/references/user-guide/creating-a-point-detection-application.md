## I. Introduction

Point detection applications in computer vision AI are helpful when it is necessary to identify and locate specific points or landmarks within images or video streams.

## II. Use-cases

The Point Detection application is used to detect output subjects in an image localized around a point. This application can be used in various fields such as manufacturing, gambling business, and medical imaging. For instance, in medical imaging it can help in identifying and locating anatomical landmarks, tumors, or other structures, which can aid in diagnosis and treatment planning.

## III. Possible issues:

However, some issues can arise while using the app. For example, blurry or poor-quality media can affect the app's performance. Similarly, media captured from different angles may also cause issues while using the app.
