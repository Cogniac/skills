# Go to a Tenant

## Tenant Overview

Once the user has successfully logged in, the experience differs based on their association with tenants.

- If the user is associated with only a single tenant, they are taken directly to that tenant's home page after logging in.

- If the user is associated with multiple tenants, they will see a list of the tenants to choose from. Any pending tenant invitations will also be displayed in this list. Users can accept invitations, which will then be added to their list of tenants, or decline them as needed.

After entering a tenant, a user associated with multiple tenants can switch the tenants. They can do this by clicking on the dropdown menu in the navigation bar, which will take them back to the tenant list. It's important to note that only users associated with multiple tenants will see the "Switch Tenant" option in the dropdown menu.
