# Release Notes — 2024

## December 20, 2024

- The media included in the feedback instructions can be opened in a new tab for a more detailed examination.

- The user can select the ‘i’ icon, which will open the Feedback Instructions modal for them. Once they select an image, they will be redirected to a new tab with the selected media.![](../../../assets/media/images/image%28179%29.png)

![](../../../assets/media/images/image%28180%29.png)Once the car from the above media list has been selected, a screen like the one below should display more media details.

![](../../../assets/media/images/image%28181%29.png)

## November 11, 2024

- Added the Segmentation application type (Limited beta release)

  - The Segmentation application will enable both instance and panoptic segmentation, allowing the users to provide masks for every instance within each class.

  - Currently, the Segmentation application is beta-ready and available upon request. You can review the full document article [here](/docs/segmentation-application)

![](../../../assets/media/images/image%28177%29.png)

- Implemented the ****Usersnap**** feature that allows users to submit their feedback on the Cogniac UI (It can include a screenshot or a video recording of the UI). Feedback can then be automatically sent to project managers, developers, or support teams to provide a detailed visual report on issues, bugs, or user experiences.

![](../../../assets/media/images/image%28178%29.png) - You can review the detailed article on the feature [here](/docs/usersnap-feature)

## November 4, 2024

- Users can add Python packages and Base Container Images for review and approval within the Integration Applications. Once they submit a new package for review, they will receive an email with the appropriate status (pending/ valid/ invalid).

![](../../../assets/media/images/image%28174%29.png)

Please find the full feature document [here](/docs/create-and-maintain-an-access-control-list-acl)

## September 10, 2024

- Implemented the Leaderboard feature to include Models of different Inference Precision, such as INT8, FP16, and FP32. Detailed information about these precision Models can be found in [this document.](/docs/leaderboard-feature-summary)

![](../../../assets/media/images/image%28169%29.png)
