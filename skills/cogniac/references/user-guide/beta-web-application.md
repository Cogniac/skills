# Cogniac Beta — Web App

This release introduces several enhancements and usability improvements designed to streamline workflows, improve monitoring, and enable more intuitive feedback collection.

### ****Accessing the Beta Web application****

🔗 ****URL:**** [beta.cogniac.io](http://beta.cogniac.io)
Existing ****Cogniac users**** can log in using their current credentials.

Users are encouraged to share UI Feedback, issues, or suggestions directly through ****Usersnap****, which is integrated into the beta environment.
![](../../../assets/media/images/image%28256%29.png)

### ****What’s New in Beta Web App****

### ****1. Enhanced Process Map****

- Introduced a ****minimap**** feature for better navigation.
- Users can ****zoom in/out**** for improved visibility and tracking of processes.

![](../../../assets/media/images/image%28238%29.png)

### ****2. Updated Apps List****

- Displays ****all applications**** available within the tenant.
- Supports sorting by:

  - Application Type
  - Performance Metrics
  - Feedback Status
    ![](../../../assets/media/images/image%28257%29.png)
- ****Enhanced Search:****
  Users can now search across:

  - EdgeFlows
  - Deployment Groups
  - Applications, Subjects, Media (existing functionality)
    ![](../../../assets/media/images/image%28258%29.png)
- ****Give Feedback:**** A shortcut button allows quick access to the ****Pending feedback queue**** for any selected application.
  ![](../../../assets/media/images/image%28260%29.png)

### ****3. EdgeFlow Enhancements****

The ****EdgeFlow**** section has been redesigned with a focus on better visibility and insights:

- Enhanced visual layout
- Additional metrics displayed for the selected duration:

  - Model Detections
  - Aggregated GPU Pixels
  - GPU Utilization
  - GPU Memory Usage
  - GPU Temperature

![](../../../assets/media/images/image%28261%29.png)

![](../../../assets/media/images/image%28262%29.png)

### ****4. Deployment Workflow Improvements****

- Added a ****Search**** feature for easy workflow discovery.
- Search by:

  - Workflow ID
  - Name
  - Deployable On
  - Deployment Group

![](../../../assets/media/images/image%28270%29.png)

### ****5. Upgraded Application Creation Flow****

The application creation process has been ****redesigned**** for clarity and efficiency, adapting intelligently to usage context. Users are now able to directly select the type of application, split into three categories (Camera App, Model App, and Logic App)

- Camera App selected
  ![](../../../assets/media/images/image%28267%29.png)

- Model App selected

![](../../../assets/media/images/image%28268%29.png)

- Logic App selected

  ![](../../../assets/media/images/image%28269%29.png)

### ****6. Model Runtime Selection****

Users can now select ****model runtime options**** during application setup, enabling more flexibility and performance optimization.

![](../../../assets/media/images/image%28266%29.png)

### ****7. Subject ID Copy Feature****

Quickly ****copy Subject IDs**** without having to open the subject page — a small but impactful productivity boost.

![](../../../assets/media/images/image%28264%29.png)

### ****8.**** Mini map navigation

We introduced a mini map that allows users to navigate through the page more easily without scrolling. It is always visible on the right-hand side, and as the user clicks on a part of it, it will open the selected section.

![](../../../assets/media/images/image%28265%29.png)

### ****9. Pixel Measurement Tool****

A new ****Pixel Measurement Tool**** enables users to measure image dimensions directly within the app, providing improved accuracy and analysis.

![](../../../assets/media/images/image%28242%29.png)
