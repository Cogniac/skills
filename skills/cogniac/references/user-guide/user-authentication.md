# User Authentication

## ****Overview****

Cogniac CloudCore supports username and password authentication for web-based access and API key-based authentication for programmatic access.

## Supported authentication types

- Individual users can enable [TOTP](https://en.wikipedia.org/wiki/Time-based_One-Time_Password) multi-factor authentication to enhance account security. This feature is compatible with popular mobile authenticator apps like Google Authenticator, Microsoft Authenticator, LastPass Authenticator, and Authy.
- SAML authentication is available for customers on the CloudCore Enterprise tier. When users enter their email on Cogniac's login page, they are redirected to their organization's access management provider (e.g., Okta). After successfully logging in through the access management provider's site, users are redirected back to the web app.
- A password expiration policy can be optionally configured for each tenant as needed.
- All authentication options can be managed from the User Settings page, which is accessible by selecting "Settings" from the main dropdown menu.
