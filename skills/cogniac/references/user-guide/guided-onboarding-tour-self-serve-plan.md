# Guided Onboarding Tour - Self-Serve Plan

This guide will show you how to start Cogniac's Onboarding Tour. It is accessible only to PayGo users. After completing it, you can create applications, upload media to subjects, and train an application.

- To access the Onboarding Tour, click the "****Help****" button and select "****Quick Tour****''. This will guide you through the steps needed to get familiar with Cogniac.

  ![](../../../assets/media/images/guided-onboarding-tour---self-serve-plan-image-s4e207ot.png)
- A window will appear once you select the "Quick Tour" button. To start the tutorial, click the "****Activate the Tutorial****" button. From there, follow the steps and read the instructions for better understanding.

![](../../../assets/media/images/guided-onboarding-tour---self-serve-plan-image-q4s70ha0.png)
